## 一、API调用成本测算

### 1.1 成本模型假设

| 参数 | 数值 | 说明 |
|-----|------|------|
| 日均对话量 | 500 轮 | 每轮 = 用户1条 + AI 1条 |
| 平均消息长度 | 200 tokens | 输入+输出平均 |
| 工作日 | 250 天/年 | 扣除周末和节假日 |
| 年增长 | 30% | 业务增长预期 |

### 1.2 LLM成本对比

**方案A: DeepSeek-V3 主力 (推荐)**
- 单价: $0.002 / 1K tokens (输入 $0.001, 输出 $0.003)
- 日均成本: 500轮 × 200 tokens × $0.002 / 1000 = **$0.2/天**
- 月成本: $0.2 × 30 = **$6/月**
- 年成本: $6 × 12 = **$72/年**

**方案B: GPT-4o**
- 单价: $0.015 / 1K tokens
- 日均成本: 500 × 200 × $0.015 / 1000 = **$1.5/天**
- 月成本: **$45/月**
- 年成本: **$540/年**

**方案C: Gemini Flash**
- 单价: $0.001 / 1K tokens
- 日均成本: 500 × 200 × $0.001 / 1000 = **$0.1/天**
- 月成本: **$3/月**
- 年成本: **$36/年**

**成本对比总结**:
| 方案 | 月成本 | 年成本 | vs DeepSeek |
|-----|--------|--------|-------------|
| DeepSeek-V3 | $6 | $72 | 基准 |
| GPT-4o | $45 | $540 | 7.5x |
| Gemini Flash | $3 | $36 | 0.5x |

**推荐**: DeepSeek-V3 主力 + Gemini Flash 降级 = **$6/月**

## 四、ROI计算模型

### 4.1 投入成本汇总

| 项目 | 年成本 (CNY) |
|-----|-------------|
| LLM API (DeepSeek) | ¥520 |
| 服务器 (GCP) | ¥2,100 |
| 其他服务 | ¥672 |
| 开发维护 (10%人工) | ¥2,430 |
| **总投入** | **¥5,722** |

### 4.2 ROI计算

```
年度节省: ¥243,000
年度投入: ¥5,722

净收益: ¥243,000 - ¥5,722 = ¥237,278

ROI = (净收益 / 投入) × 100%
    = (237,278 / 5,722) × 100%
    = 4145%

投资回报周期: 5,722 / (243,000/12) = 0.28 个月 ≈ 9 天
```

**结论**: 投资9天即可回本，年度ROI超过4100%
